# 🧮 Calculadora em Python

Uma calculadora simples com interface gráfica desenvolvida em Python utilizando a biblioteca PySimpleGUI.

---

## 🚀 Funcionalidades

✔ Interface moderna
✔ Operações básicas (+, -, ×, ÷)
✔ Design organizado
✔ Fácil de usar

## 🛠 Tecnologias utilizadas

* Python
* PySimpleGUI

---

## ▶️ Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/JailsonSales/Calculadora.git

# Entre na pasta
cd Calculadora

# Instale as dependências
pip install PySimpleGUI

# Execute o projeto
python calculadora.py
```
